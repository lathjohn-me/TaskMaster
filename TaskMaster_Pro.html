Welcome to TaskMaster Pro

Connect Google Drive to sync tasks across all your devices. One-time setup — all in your browser.

1
Go to console.cloud.google.com and sign in with your personal Google account.
2
Click Select a project → New Project. Name it TaskMaster and click Create.
3
Search Google Drive API and click Enable.
4
Go to Credentials → Create Credentials → OAuth client ID. Choose Web application. Add this file's URL as an Authorised JavaScript origin. Copy the Client ID.
5
Go to OAuth consent screen, set External, add your email as a test user.

Or skip for now — tasks save locally only.
